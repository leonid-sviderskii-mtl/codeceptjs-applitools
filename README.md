### General setup:
1. Install webdriver manager: `npm install webdriver-manager -g`
2. Run update: `webdriver-manager update`
3. Launch Webdriver: `webdriver-manager start`

### To run Applitools on Lamdatest:
1. Create `.env` file
2. Provide credentials for:
    `LAMBDATEST_USERNAME`
    `LAMBDATEST_KEY`
    `APPLITOOLS_KEY`

### To run the tests:
`npm run test`
